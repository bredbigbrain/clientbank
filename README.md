# clientbank
educational project sibsutis RI48

Модуль формата BlackBox
Шаблон ака MVC

Данные недоступны снаружи
Уникальный ID, Фио
Кол-во денежных средств
История платежей с возможностью отмены
Добавление новых клиентов
Сохранение и загрузка XML базы

Мы создаем программу, которая имеет функции:

1. Добавление новых клиентов со всех инфой о них.

2. Перевод средств со счёта на счёт, который использует функции:
3. Начисление средств на счёт клиента
4. Снятие средств со счёта клиента

5. Отмена перевода:
6. Выгрузить список транзакций

7. Выдача кредитов с проверкой клиента
8. Платежи по кредиту

9. Бот-ассистент
10. Напоминание о переводах
